Tipos de datos primitivos de Java

Tipo int

Es un tipo de datos numerico, entero de 32 bits con signo.
Su rango va desde -2e31 hasta 2e31-1.

Tipo float

Es un tipo de datos para almacenar numeros como coma flotante de precision 64 bits.
